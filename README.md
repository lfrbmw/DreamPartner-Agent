# 梦幻搭子 Agent

## 项目简介

"梦幻搭子 Agent" 是一款专注于情感支持与陪伴的AI智能体应用，旨在为用户提供温暖的关怀和情感慰藉，让用户在快节奏的生活中找到一个可以倾诉和陪伴的虚拟伙伴。

## 核心功能

### 1. 情感倾听与回应
- 基于GLM-4.6大语言模型，提供高质量的情感对话体验
- 能够理解用户情绪并给予温暖、贴心的回应
- 支持多轮对话，保持上下文连贯性

### 2. 个性化性格定制
- 提供四种预设性格类型：
  - 温柔体贴：像春风般温暖，细腻关怀你的情绪
  - 幽默风趣：用轻松幽默化解你的烦恼
  - 成熟稳重：理性分析，给你可靠建议
  - 活泼可爱：充满活力，带来阳光般的好心情
- 用户可根据个人喜好自由切换搭子性格

### 3. 聊天记录管理
- 自动保存聊天历史记录
- 支持查看历史对话
- 提供记录导出功能（JSON格式）
- 一键清空聊天记录

### 4. 移动端优化体验
- 响应式设计，完美适配移动设备
- 精心设计的UI界面，营造温馨治愈的交互体验
- 流畅的动画效果和交互反馈

## 技术实现

### 前端技术栈
- React 18：构建用户界面的核心框架
- Ant Design 5：UI组件库，提供美观易用的界面组件
- React Router v6：实现页面路由导航
- Less：CSS预处理器，实现模块化样式管理
- localStorage：本地数据存储

### AI服务集成
- 集成智谱AI GLM-4.6大模型API
- 自定义系统提示词，实现不同性格特征的AI角色
- 支持思维链推理（thinking enabled）

### 项目特色
1. **情感化设计**：采用温暖的配色方案（蜜桃色系）和圆润的界面元素，营造温馨治愈的视觉体验
2. **模块化架构**：清晰的项目结构，便于维护和扩展
3. **本地化存储**：所有数据均存储在用户本地，保护隐私安全
4. **完整功能闭环**：涵盖对话、设置、历史记录查看等完整功能流程

## 使用场景
- 情绪低落时的情感倾诉对象
- 日常生活中的陪伴伙伴
- 工作学习中的减压工具
- 创意灵感的交流对象
